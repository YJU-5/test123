# Gemini CLI 시연용 빠른 명령어 모음

## 🚀 환경 설정
export GOOGLE_API_KEY="YOUR_API_KEY"
gemini --version

## 🎬 1. 동영상 제작 자동화
gemini -p "유튜브 영상 '2024년 최고의 생산성 도구 5가지'에 대한 5분짜리 스크립트를 작성해줘. 서론, 본론(각 도구별 설명), 결론을 포함하고, 각 섹션에 예상 시간을 표시해줘. 말투는 활기차고 친근하게."

## 📄 2. 업무 문서 자동화
gemini -p "$(cat meeting_minutes.txt) 위 회의록에서 논의된 핵심 결정사항과 다음 액션 아이템을 요약해줘."

## 💻 3. 코드 생성 자동화
gemini -p "CSV 파일에서 'sales' 컬럼의 평균을 계산하는 Python 코드를 작성해줘. pandas 라이브러리를 사용하고 주석을 상세하게 달아줘."

## 🎨 4. 창의적 프롬프트 아이디어 생성
gemini -p "미드저니에서 쓸, '신비로운 숲속에 있는 고대 유적'에 대한 5가지 상세한 프롬프트 아이디어를 알려줘. (예: 사진작가, 애니메이션 스타일 등)"

## 🔧 추가 유용한 명령어들

### 이메일 작성
gemini -p "고객에게 제품 배송 지연에 대한 사과 이메일을 작성해줘. 전문적이면서도 친근한 톤으로."

### 회의 안건 작성
gemini -p "다음 주 팀 미팅 안건을 작성해줘. 프로젝트 진행상황, 이슈사항, 다음 단계 계획을 포함해서."

### 소셜미디어 포스트
gemini -p "회사 신제품 출시를 알리는 인스타그램 포스트를 작성해줘. 해시태그 5개 포함."

### 코드 리뷰
gemini -p "$(cat your_code.py) 이 Python 코드를 리뷰해주고 개선점을 제안해줘."
